## `relate app:launch [APPID]`

```
USAGE
  $ relate app:launch [APPID]

OPTIONS
  -D, --dbmsId=dbmsId            (required)
  -H, --host=host                [default: http://localhost:3000]
  -P, --principal=principal      (required)
  -e, --environment=environment  Environment to run the command against
```
